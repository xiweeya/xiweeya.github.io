Making Text-align Justify Working

一、问题

在页面上设置了父元素：width: 100%; text-align: justify;

设置了子元素：display: inline-block;

在浏览器（Chrome 中）自动扩展对齐无法实现

二、解决方法

参考：http://stackoverflow.com/questions/6228347/text-align-justify-not-working

为父元素添加宽度为一整行的空白后缀元素：

.parent{
  width: 100%;
  text-align: justify;
}
.parent:after{
  content: "";
  display: inline-block;
  width: 100%;
}
.children{
  display: inline-block;
}

三、应用

1、两边分散对齐的表单 label

效果图表单效果为：

上图中表单标签和输入框为一组，每组宽度固定，其中标签、输入框长度也固定，标签文字对齐方式为两端对齐。因此标签文字的部分很容易联想到可以用text-align:justify;实现。

标签部分实现思路为，以最大的标签文字数设置标签部分的固定宽度，然后使用text-align:justify;让其分散并两端对齐。

label {
  width: 4em;
  text-align: justify;
}
label:after{
  content: "";
  display: inline-block;
  width: 100%;
}

由于添加了一整行的:after，文字变成了两行，将与冒号和输入框不在统一水平线上，如下图，

为了保持元素显示在统一水平线上，用vertical-align设置水平位置统一，即

label {
  width: 4em;
  text-align: justify;
  vertical-align: top;
}
label:after{
  content: "";
  display: inline-block;
  width: 100%;
}

之后按效果图调整好间距、加入冒号即可。
